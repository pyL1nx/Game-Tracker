# Game-Tracker

Game Tracker is a lightweight Windows application that records and displays total playtime and session counts for any selected game executable. It runs in the background, detects when games start and stop, and maintains accurate local statistics without requiring any online services or accounts.

This repository provides the compiled Windows build for end-users.

# Features
Automatic Playtime Tracking

Detects when added games start and stop running.

Records total playtime and session counts.

Updates statistics in real time.

Modern User Interface

Full-width game cards with extracted high-resolution game icons.

Light and dark theme modes.

Search functionality.

Favorite sorting system.

Background Operation

Minimizes to the system tray.

Continues tracking while minimized.

Can be reopened or fully closed through tray controls.

Local Data Storage

All data is saved on the user's machine.

No internet connection or account required.

# Download

The latest Windows build is available in the Releases section of this repository.

Download the ZIP archive, extract it, and launch:

main.exe


Do not remove any other files from the extracted folder.
They are required for the application to run.

How to Use

Download and extract the release ZIP.

Run main.exe.

Click Add Game and select a game's .exe file.

The tracker will automatically start monitoring whenever that game is launched.

Close the window to minimize the application to the system tray.

Right-click the tray icon to reopen or quit.

Playtime and session history remain saved even if the game is uninstalled or temporarily removed.

System Requirements

Windows 10 or Windows 11

No additional installations required
